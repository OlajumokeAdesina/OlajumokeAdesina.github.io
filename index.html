Dengue Virus Project

A study to demonstrate computational drug discovery by implementing Machine Learning techniques in the drug discovery process. The ML models used are the Support Vector Machine and Binary Logistic Regression models, which were used to predict the bioactivity class of the molecules using Lipinski Descriptors. Check out my awesome work.
The software used for this project is Python programming language in Google Colaboratory which is built on top of Jupyter Notebook. The research conducted in this project involves the analysis of secondary data gotten from open-source databases ChEMBL and Kaggle. The ChEMBL dataset was accessed 12th August 2022 and the Kaggle dataset was accessed 28th August 2022.
